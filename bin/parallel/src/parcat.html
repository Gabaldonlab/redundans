NAME

parcat - cat files or fifos in parallel

SYNOPSIS

parcat file(s)

DESCRIPTION

GNU parcat reads files or fifos in parallel. It writes full lines so there will be no problem with mixed-half-lines which you risk if you use:

   (cat file1 & cat file2 &) | ...

EXAMPLES

Do be done

  mkfifo slot-{1..5}-digit-{0..9}
  parallel -j5 'seq 100000 | grep {} > slot-{%}-digit-{}' ::: {0..9} &
  parallel parcat slot-{1..5}-digit-{} '>' digit-{} ::: {0..9}
